Lors de la configuration d'une nouvelle intégration HubSpot à l'aide des API REST, il peut être difficile de tester et de confirmer que les données de visionnage vidéo sont bien transmises à HubSpot. Dans cette rubrique, vous apprendrez comment tester et confirmer systématiquement que l'intégration fonctionne et que les données de visualisation et les prospects sont envoyés à HubSpot.

Remarque description: Cette rubrique permet de vérifier les connexions HubSpot établies à l'aide des API REST.
Remarque: Pour plus d'informations sur la configuration du module Campaign pour une utilisation avec HubSpot, voir Intégration de Video Cloud à HubSpot à l'aide des API REST.

Avant de commencer

Avant de valider l'intégration, on suppose que vous avez:

  • Intégration réussie de Brightcove Campaign à HubSpot
  • Création d'au moins un lecteur compatible avec la campagne et configuration pour utiliser la connexion HubSpot

Configuration d'un formulaire de pistes de test et publication d'une vidéo

À des fins de test, vous devrez créer et configurer un formulaire de prospect personnalisé dans Campaign. Il s'agit d'un formulaire HubSpot dont vous avez collé le code intégré dans Campaign. Même si vous n'utilisez pas de formulaires de prospects dans votre environnement de production, ce formulaire est utilisé pour être sûr que vous êtes bien connu dans votre base de données et sur la page. Pour créer un formulaire HubSpot et configurer Campaign pour l'utiliser, procédez comme suit.

  1. Connectez-vous à votre compte HubSpot.
  2. Créez et personnalisez un formulaire HubSpot. Pour plus d'informations sur la création d'un formulaire HubSpot, consultez Création de formulaires de prospects personnalisés pour HubSpot.
  3. Cliquez sur Partager en haut de la page.
  4. Copiez le code intégré du formulaire dans le presse-papiers.
  5. Revenez au module Campaign.
  6. Créez un nouveau formulaire pour prospects à l'aide du code intégré du formulaire HubSpot.
  7. Définissez votre formulaire personnalisé pour qu'il affiche le formulaire au bout de 5 secondes. De cette façon, vous saurez exactement quand il doit apparaître.
  8. Cochez la case pour Toujours afficher le formulaire du prospect, même s'il est déjà connu. Cela permet de s'assurer que le formulaire continue de s'afficher après l'avoir rempli pendant le test.
  9. Enregistrez le formulaire.
  10. Configurez le lecteur Brightcove avec la connexion HubSpot pour utiliser le formulaire de prospects HubSpot qui vient d'être créé.

Générer un lien d'aperçu pour visionner la vidéo

Pour générer un lien d'aperçu vers la vidéo, procédez comme suit.

  1. Ouvrez le module Médias.
  2. Sélectionnez une vidéo et publiez-la sur un lecteur Web. Assurez-vous de sélectionner le lecteur compatible avec la campagne avec le formulaire de prospect.
  3. Cliquez sur le lien d'aperçu pour ouvrir la vidéo dans une nouvelle fenêtre de navigateur. Les tests de cette manière éliminent toutes les variables potentielles qui pourraient être introduites par les lecteurs intégrés à une page HTML.
  4. Vérifiez que la vidéo est lue et que le formulaire de prospect s'affiche au bout de 5 secondes.
  5. Remplissez le formulaire de prospect en utilisant des informations de contact et une adresse e-mail uniques (c'est-à-dire le nom d'une célébrité ou un nom contenant les mots â€œbctestâ €-fr/) afin de le retrouver facilement plus tard dans HubSpot. Cela élimine toutes les variables de niveau navigateur, lecteur ou base de données. En remplissant le formulaire en tant que nouvel utilisateur, nous pouvons être sûrs que vous êtes désormais un utilisateur connu de HubSpot.

Confirmer que Campaign a réussi à capturer les données

Avant de rechercher les données de visionnage des vidéos dans HubSpot, vérifiez d'abord que Campaign a bien capturé la vue vidéo. Notez que l'affichage des données d'affichage dans l'interface utilisateur de la campagne peut prendre jusqu'à 3 heures. Pour confirmer que Campaign a réussi à capturer les données, procédez comme suit.

  1. Ouvrez le module Campaign.
  2. Cliquez sur Toutes les activités récentes dans le volet de navigation de gauche.
  3. Vérifiez qu'il existe un événement avec un horodatage récent indiquant le nom de la vidéo qui vient d'être visionnée. Le statut de l'événement ne doit pas être traité. Cela indique que Campaign a capturé la vue mais n'a pas encore tenté de la synchroniser avec HubSpot.
  4. Pour déclencher manuellement une synchronisation entre Campaign et HubSpot, cliquez sur Activité de synchronisation dans le menu de navigation de gauche.
  5. Dans la liste déroulante Sélectionner une connexion, sélectionnez votre intégration HubSpot. Un bouton Tester la synchronisation doit apparaître.
  6. Cliquez sur Tester la synchronisation pour déclencher une synchronisation manuelle. Notez que vous êtes limité à la fréquence à laquelle vous pouvez déclencher une synchronisation manuelle. Par défaut, Campaign synchronise les vues avec HubSpot une fois par heure.
  7. Cliquez sur Toutes les activités récentes dans le volet de navigation de gauche.
  8. Le statut de votre événement View sera mis en file d'attente. La synchronisation des données avec HubSpot peut prendre environ 15 minutes. Une fois les données synchronisées, l'état passe à l'un des deux statuts suivants:
    • synchronisé - Indique que l'utilisateur était un utilisateur connu et que les données ont été correctement transmises à son journal d'activité. Les données devraient également être visibles dans HubSpot en quelques minutes.
    • unknown_contact - Indique que l'utilisateur était considéré comme inconnu et que HubSpot n'a pas pu accepter les données. Si vous obtenez le statut unknown_contact, vérifiez que vous avez publié la vidéo en utilisant le bon lecteur, que vous avez un formulaire HubSpot en place et que vous effectuez des tests à l'aide du lien d'aperçu, et non intégré à une page.

Confirmer que les données de visionnage des vidéos se trouvent dans HubSpot

Une fois que Campaign affiche un statut synchronisé, suivez ces étapes pour vérifier que les données se trouvent dans HubSpot.

Remarque: Il faut compter entre 5 et 10 minutes pour que l'interface utilisateur HubSpot soit mise à jour avec les nouvelles données reçues de Campaign.
  1. Connectez-vous à votre compte HubSpot.
  2. Dans HubSpot, accédez à la zone Contacts et recherchez le nom unique ou l'adresse e-mail que vous avez utilisé lorsque vous avez rempli le formulaire de prospect.
  3. Cliquez sur le contact pour ouvrir le journal d'activité.
  4. Cliquez sur la liste déroulante Filtrer l'activité (en haut de la page sous le menu Activité) et vérifiez que Brightcove Video Connect est cochée. Cela garantit que l'activité de visionnage de vidéos est affichée dans le journal d'activité.
  5. Dans le journal d'activité, recherchez une activité Brightcove. Développez les détails pour afficher les données de l'événement de visualisation de la vidéo.

HubSpot exige que les données transmises via leur API soient attribuées à un contact connu. HubSpot n'accepte que les données des contacts connus. Les données des contacts inconnus ne seront pas synchronisées. Si vous n'utilisez pas de formulaires de prospects, il n'y a aucun moyen pour un utilisateur inconnu de se convertir en utilisateur connu et aucune donnée ne sera donc synchronisée.
