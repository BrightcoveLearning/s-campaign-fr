Le module Campaign permet de suivre l'activité de visionnage. Dans ce scénario, les données de visionnage ne sont pas envoyées à une plateforme d'automatisation du marketing. Un cas d'utilisation courant pour cela serait les vidéos de formation internes. Le suivi des campagnes peut être utilisé pour suivre l'activité de visionnage des employés internes. Le module Campaign stockera jusqu'à 500 000 vues d'événements par connexion. Une fois la limite atteinte, il commencera à purger les anciens événements.

Pour utiliser le suivi des campagnes, les étapes suivantes sont nécessaires:

  1. Configuration des paramètres de suivi de la campagne
  2. Configuration d'un formulaire de lecteur et de prospect pour le suivi des campagnes
  3. Afficher les données vidéo

Configuration des paramètres de suivi des campagnes

Pour configurer les paramètres de suivi de la campagne, procédez comme suit:

  1. Ouvrez le module Campaign.
  2. Dans la navigation de gauche, cliquez sur Connexions de données.
  3. Localisez la section Suivi des campagnes et cliquez sur Afficher.
  4. Par défaut, le suivi des campagnes est activé pour tous les comptes. Modifiez les paramètres selon vos besoins:
    • Nom de la connexion: identifie la connexion
    • État de la connexion: détermine si les données d'engagement vidéo de Video Cloud sont enregistrées par le module Campaign.
      • Activé - Les données sont collectées
      • Désactivé - Aucune donnée n'est collectée (la connexion est désactivée)
    • Track Data for Unknown Viewers - Détermine si les données de suivi sont enregistrées pour des spectateurs inconnus. Pour identifier les spectateurs lors de l'utilisation du suivi de campagne, un formulaire de prospect de campagne peut être utilisé.
      Remarque: Avec le suivi des campagnes, jusqu'à 1 000 spectateurs seront suivis par période de 24 heures.
    • Pendant combien de temps Brightcove doit-elle conserver les données ? - La politique de conservation des données détermine la durée pendant laquelle les données de visionnage et les données du formulaire de prospect (si vous utilisez un formulaire de prospect Campaign) seront conservées dans votre compte Video Cloud. Vous devez ajuster les paramètres de conservation conformément à votre politique de confidentialité. Il n'est pas nécessaire de conserver les données utilisateur dans Video Cloud.

Configuration d'un formulaire de lecteur et de prospect pour le suivi des campagnes

L'activité de visionnage ne peut être suivie que lorsque les vidéos sont visionnées dans un lecteur vidéo compatible avec la campagne. Vous devez utiliser un lecteur compatible avec la campagne lorsque vous utilisez le suivi de campagne. Les formulaires de prospects peuvent être affichés pendant la lecteur de la vidéo pour capturer les informations du spectateur. Les formulaires de prospects ne sont pas nécessaires lorsque vous utilisez le suivi de campagne.

Pour configurer le suivi des campagnes avec un lecteur compatible avec la campagne et un formulaire de prospects, procédez comme suit:

  1. Cliquez sur Joueurs de campagne dans le menu de navigation de gauche.
  2. Déterminez le lecteur compatible avec la campagne à utiliser. S'il n'existe aucun lecteur compatible avec la campagne, cliquez sur Créer un lecteur pour en créer un. Pour plus d'informations sur la création de lecteurs compatibles avec la campagne, consultez Création d'un lecteur compatible avec la campagne.
  3. Bien qu'un formulaire de prospect ne soit pas requis, les formulaires de prospects peuvent être utilisés pour collecter des informations sur le spectateur et transformer des spectateurs anonymes en spectateurs connus. Cliquez sur Lead Forms et créez un formulaire pour prospects. Pour plus d'informations sur la création de formulaires de prospects, voir Création d'un formulaire de prospect de campagne.
  4. Cliquez sur Campaign Players.
  5. Localisez le lecteur compatible avec la campagne à utiliser. Définissez la valeur Se connecter à pour le lecteur sur Suivi de campagne.
  6. Si un formulaire de prospect est utilisé, sélectionnez-le dans la liste déroulante Lead Form.

Visualisation des données vidéo

Plusieurs options sont disponibles pour visualiser les données vidéo lors de l'utilisation du suivi de campagne:

  • Viewer Activities (Events) - Fourni dans le cadre du module Campaign, cette option peut être utilisée pour afficher les données vidéo dans le module Campaign. Pour plus d'informations, consultez la section Affichage des événements des lecteur s dans le module Campaign.
  • Profils des spectateurs - Fourni dans le cadre du module Campaign, cette option peut être utilisée pour afficher les données vidéo en fonction d'une valeur de profil utilisateur dans le module Campaign. Pour plus d'informations, consultez la section Affichage des informations sur le profil
  • Exporter les données - Fourni dans le cadre du module Campaign, cette option peut être utilisée pour exporter les données de visionnage de vidéos et de prospects. Pour plus d'informations, voir Exportation des données de campagne à partir du module Campaign.
  • API Audience - L'API Audience peut être utilisée pour récupérer par programme les données de visionnage de vidéos et de prospects. Pour plus d'informations, voir Vue d'ensemble: API Audience.
