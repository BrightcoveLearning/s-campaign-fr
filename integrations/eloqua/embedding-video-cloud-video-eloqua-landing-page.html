Le module Campaign permet d'ajouter une vidéo Video Cloud aux pages de destination Eloqua en tant que type de contenu Cloud. La vidéo est ajoutée à une page de destination depuis l'éditeur WYSIWYG de la page de destination Eloqua.

Pour ajouter une vidéo à une page d'accueil Eloqua, les étapes suivantes doivent être effectuées.

  1. Créer une page de destination dans Eloqua
  2. Sélectionnez une vidéo et un lecteur

Création d'une page de destination Eloqua

Les pages de destination Eloqua peuvent être utilisées comme destination pour les visiteurs qui cliquent sur les e-mails envoyés par Eloqua. Pour créer une page de destination, procédez comme suit.

Remarque: Pour obtenir des informations complètes sur les pages de destination Eloqua, consultez la documentation Oracle Eloqua.
  1. Connectez-vous à votre compte Video Cloud. Sélectionnez le compte sur lequel figure la vidéo que vous souhaitez utiliser sur la page de destination. Ce compte doit également avoir accès au module Campaign.
  2. Connectez-vous à votre compte Eloqua.
  3. Accédez à Ressources > Pages de destination, puis cliquez sur Créer une page de destination.
  4. Cliquez sur Page de destination HTML vierge, puis sur Choisir.
  5. Dans le volet de navigation de gauche, cliquez sur Cloud Content. La boîte de dialogue Cloud Content s'ouvre.
  6. Faites glisser l'objet Brightcove Video sur la page HTML et déposez-le entre les balises <body >.
    Remarque: Plusieurs lecteur s peuvent être ajoutés à une page de destination en faisant glisser et en déposant un lecteur entre les balises <body >. Enregistrez la page de destination après avoir ajouté chaque lecteur .

La boîte de dialogue Cloud Content Configuration s'ouvre et vous permet de sélectionner une vidéo et un lecteur.

Sélection d'une vidéo et d'un lecteur

La boîte de dialogue Cloud Content Configuration permet de sélectionner un compte, une vidéo et un lecteur Video Cloud.

  1. Utilisez la liste déroulante en haut à droite pour sélectionner le compte Video Cloud à utiliser. Une fois qu'un compte est sélectionné, la boîte de dialogue s'actualise pour afficher les vidéos et les lecteurs du compte sélectionné.
  2. Sélectionnez une vidéo à afficher sur la page de destination. Les 20 vidéos les plus récentes apparaîtront dans la liste.
  3. Sélectionnez un lecteur à utiliser. Seuls les lecteur s ayant activé la campagne apparaîtront.
  4. Définissez la taille du lecteur.
  5. Cliquez sur Enregistrer pour enregistrer les modifications.
  6. Cliquez sur X pour fermer la boîte de dialogue Cloud Content Configuration. Le code HTML sera mis à jour avec le code pour intégrer le lecteur.
  7. (Facultatif) Ajoutez d'autres objets à la page de destination.
  8. Cliquez sur Enregistrer pour enregistrer la page de destination.
  9. Donnez un nom à la page de destination, puis cliquez sur Enregistrer.
