Les événements View sont enregistrés lorsque les vidéos sont visionnées à l'aide de lecteurs compatibles avec la campagne. Les données relatives aux prospects sont capturées lorsque les utilisateurs saisissent des données dans les formulaires de prospects Le module Campaign propose deux types de formulaires de prospects:

  • Formulaires de pistes de campagne - Ces formulaires sont créés en sélectionnant parmi un ensemble de champs prédéfinis dans le module Campaign. Ces formulaires enregistrent les données des prospects dans Video Cloud. Pour plus d'informations sur la création de formulaires de pistes de campagne, voir Création d'un formulaire de prospect de campagne
  • Formulaires de prospects personnalisés - Ces formulaires sont créés à l'aide de votre plateforme d'automatisation du marketing. Le code intégré du formulaire est ensuite utilisé dans le module Campaign pour créer un formulaire personnalisé pour les prospects. Ces formulaires envoient les données des prospects à la plateforme d'automatisation du marketing. Pour plus d'informations sur la création de formulaires de prospects, consultez le document correspondant à votre plateforme marketing:
    • Eloqua
    • Marketo
    • Salesforce
    • HubSpot

Lors de l'exportation des données de prospects à partir du module Campaign, seules les données capturées à l'aide des formulaires de prospects de campagne seront Les données des prospects provenant de formulaires de prospects personnalisés seront stockées dans votre logiciel d'automatisation du marketing.

Pour exporter des données, procédez comme suit:

  1. Ouvrez le module Campaign.
  2. Cliquez sur Exporter les données de la campagne dans le menu de navigation
    Remarque: Vous devez être administrateur de compte pour accéder au menu Exporter les données.
  3. Sélectionnez une connexion de données à partir de laquelle exporter les données. La liste déroulante Connexion aux données répertorie toutes les intégrations qui ont été configurées.
  4. Sélectionnez les données à exporter.
    • Afficher les événements - enregistrés par les lecteur s ayant activé la campagne
    • Leads - Capturé à l'aide de formulaires
  5. Utilisez le sélecteur de dates pour sélectionner une date de début et une date de fin.
  6. Sélectionnez un format d'exportation.
    • Valeurs séparées par des virgules (CSV)
    • Excel (XLS)
  7. Cliquez sur Exporter pour télécharger les données des prospects.
  8. Les données de l'événement Lead Form et View seront téléchargées dans des fichiers séparés. Les données des pistes contiendront les données saisies dans le formulaire de prospect ainsi que les champs suivants:
    • ExternalId - Il s'agit de l'ID externe qui apparaît sur la page Toutes les activités récentes. Dans le cas d'Oracle Eloqua, il s'agit du guid Eloqua. Avec Marketo, c'est le guide Marketo
    • PlayerID: ID du lecteur utilisé lors de l'affichage du formulaire
    • VideoID: identifiant vidéo de la vidéo en cours de lecteur lorsque le formulaire était affiché
    • PageUrl: URL de la page qui affichait le formulaire de prospect (expériences Gallery uniquement)
    • CreateDAT: date et heure auxquelles les données ont été capturées
    • experience: expérience Gallery qui a collecté les données (expériences Gallery uniquement)

    Les données d'affichage des événements contiendront les champs suivants:

    • ExternalId - Il s'agit de l'ID externe qui apparaît sur la page Toutes les activités récentes. Dans le cas d'Oracle Eloqua, il s'agit du guid Eloqua. Avec Marketo, c'est le guide Marketo
    • VideoID: identifiant vidéo de la vidéo
    • VideoName: nom de la vidéo
    • watched: pourcentage de vidéos regardées (au pourcentage le plus proche)
    • TimeWatched: temps total regardé
    • PlayerID: ID du lecteur utilisé pour regarder la vidéo
    • PageUrl: URL de la page contenant le lecteur
    • CreateDAT: date et heure auxquelles les données ont été capturées
    • UpdateDAT: date et heure auxquelles l'événement a été enregistré pour la dernière fois, ce qui correspond approximativement au moment où le spectateur a arrêté de regarder ; si la vidéo ne dure qu'une minute, CreateDAT et UpdateDAT seront très proches
    • IsSynced - Spécifie si cet événement de vue a été automatiquement exporté vers l'outil d'automatisation du marketing de l'éditeur™
    • TrackingID - ID de suivi transmis en tant que paramètre d'URL. Pour plus d'informations sur l'utilisation d'un identifiant de suivi, voir Création d'intégrations personnalisées avec le module Campaign.
    • Champs UTM - UTMSource, UTMMedium, UTMCampaign, UTMTerm et UTMContent. Pour plus d'informations sur l'utilisation des champs UTM, voir Utilisation des champs UTM dans le module Campaign.
    • metric1-3, event1-3 - Utilisé lors de l'utilisation de métriques et d'événements personnalisés
