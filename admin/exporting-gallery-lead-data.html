Les données du formulaire de prospect Gallery sont collectées lorsqu'une expérience Gallery a été configurée pour utiliser un formulaire de prospect. Pour plus d'informations sur la configuration d'une expérience de portail pour collecter des données de prospects, consultez la section Utilisation d'un formulaire de prospect avec une expérience de portail

Lors de l'exportation de prospects Gallery à partir du module Campaign, seules les données capturées à l'aide des formulaires de pistes de campagne seront expor Les données des prospects provenant de formulaires de prospects personnalisés seront stockées dans votre logiciel d'automatisation du marketing.

Remarque: Seuls les utilisateurs disposant de privilèges d'administrateur seront en mesure de télécharger les données des prospects.

Pour exporter les données des prospects Gallery, procédez comme suit.

  1. Ouvrez le module Campaign.
  2. Dans le menu de navigation de gauche, cliquez sur Exporter les pistes de galerie
  3. Sélectionnez une expérience Gallery à partir de laquelle exporter les prospects.
  4. Entrez une date de début et une date de fin pour les prospects.
  5. Sélectionnez un format d'exportation, Valeurs séparées par des virgules ou Excel.
  6. Cliquez sur Exporter pour télécharger les données des prospects.

Les champs suivants seront inclus dans le fichier exporté:

  • Tous les champs de formulaire de prospects possibles
    • Adresse e-mail
    • Prénom
    • Nom
    • Téléphone professionnel
    • pays
    • Nom de l'entreprise
    • industrie
  • ExternalId: identifiant utilisé pour identifier de manière unique le spectateur
  • PlayerID: ID du lecteur qui affichait le formulaire de prospect
  • VideoID: identifiant vidéo de la vidéo visionnée lors de la collecte des données
  • PageUrl - La visionneuse d'URL de page était activée lorsque les données ont été
  • CreateDAT - date et heure de collecte des données
  • experience: nom de l'expérience affiché lors de la collecte des données
