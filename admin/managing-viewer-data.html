L'option Gérer les données des visionneuses vous permet de rechercher un utilisateur, puis de le supprimer ou d'exporter les données de cet utilisateur.

Remarques:
  • En général, lors de l'intégration à une plate-forme d'automatisation du marketing (intégration MAP), le paramètre de période de conservation des données est utilisé pour contrôler la durée de conservation des données
  • Vous devez être administrateur de compte pour accéder au menu Gérer les données des visionneuses

Pour gérer les données des spectateurs, procédez comme suit.

  1. Ouvrez le module Campaign.
  2. Cliquez sur Gérer les données des visionneuses dans le volet de navigation
  3. Saisissez l'utilisateur à rechercher. Il s'agit généralement de l'une des valeurs suivantes:
    • E-mail: lors de l'utilisation du suivi de campagne et qu'un formulaire de prospect de campagne est utilisé ou avec des intégrations SSO
    • ID utilisateur: lors de l'utilisation de l'authentification unique ou des intégrations personnalisées
    • GUID personnalisé - Pour les utilisateurs anonymes
    • GUID externe: lorsqu'une intégration MAP est utilisée
  4. Si une correspondance est trouvée, sélectionnez une action:
    • Supprimer les données de visionneuse - Supprimer les données de visionnage et de prospects pour l'utilisateur
    • Exporter les données de visionneuse - Exportez le prospect et affichez les données d'événement pour l'utilisateur. Les données du lead contiendront les données saisies dans un formulaire de prospect de campagne. Les données des événements de vue contiennent tous les événements de vue pour la visionneuse.
