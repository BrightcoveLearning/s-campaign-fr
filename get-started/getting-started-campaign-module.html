Le module Campaign est utilisé pour suivre les données d'engagement vidéo pour les vidéos visionnées dans un lecteur Brightcove compatible avec Campaign. Les données de visionnage sont stockées dans Video Cloud et peuvent éventuellement être synchronisées avec des plateformes d'automatisation marketing populaires telles que Eloqua, Marketo, HubSpot et Salesforce.

Pour accéder au module Campaign, connectez-vous à Video Cloud et cliquez sur Campaign dans la navigation principale. La page d'accueil de la campagne affiche les données d'engagement vidéo qui fournissent des informations sur la durée de visionnage des vidéos, les vidéos qui ont été regardées et qui les a regardées. Cela vous permet de mieux engager et convertir votre public.

La page d'accueil permet de:

  • Afficher les performances vidéo
  • Afficher les performances des prospects
  • Afficher le statut du module Campaign

Le module Campaign permet de:

  • Configuration des connexions de données
  • Créer des lecteur s de campagne
  • Créer et gérer des formulaires de prospects
  • Publier sur e-mail
  • Afficher les événements des lecteur s
  • Afficher les profils des spectateurs
  • Afficher l'activité de synchronisation
  • Configuration des paramètres de campagne
  • Données de campagne d'exportation
  • Exporter les pistes de
  • Gérer les données des spectateurs

Visualisation des performances vidéo

La section Performances vidéo fournit des performances vidéo de haut niveau pour les vidéos qui ont été lues à l'intérieur de lecteurs compatibles avec Campaign.

Remarque: Le système d'analyse Campaign est distinct du système d'analyse général de Video Cloud, et les deux suivent les données différemment. Par conséquent, vous constaterez des différences dans certains indicateurs si vous comparez les deux. Par exemple, le système d'analyse Video Cloud compte les vues dès le démarrage d'une vidéo, tandis que le système d'analyse Campaign ne compte pas les vues avant le début de la vidéo 10 secondes. Ainsi, si 14 spectateurs démarrent la vidéo, mais que 6 d'entre eux l'arrêtent avant que 10 secondes ne se soient écoulées, les analyses de Video Cloud afficheront 14 vues, tandis que le module Campaign en affichera 8.

Commencez par sélectionner une période pour les graphiques. Les options suivantes sont disponibles:

  • 7 jours
  • 30 jours
  • 90 jours

Les graphiques suivants s'affichent:

  • Vidéos les plus vues
  • Vidéos les plus engageantes
  • Tendances des spectateurs

Vidéos les plus vues

Le graphique Vidéos les plus vues affiche les vidéos les plus vues dans les lecteurs compatibles avec la campagne pour la période sélectionnée.

Vidéos les plus engageantes

Le graphique Vidéos les plus engageantes affiche les vidéos les plus attrayantes des lecteurs compatibles avec la campagne pour la période sélectionnée. L'engagement est le pourcentage moyen de personnes regardées. Pour en savoir plus sur la façon dont nous calculons l'engagement vidéo, regardez la vidéo de formation Calculer le score d'engagement vidéo

Tendances des spectateurs

Le graphique Viewer Trends affiche les vues des vidéos au fil du temps pour la période sélectionnée. Les vues des spectateurs connus et anonymes sont affichées. Le survol du graphique affichera les points de données réels.

Remarque: Lorsque vous utilisez une connexion Marketo Munchkin, le graphique Viewer Trends affiche uniquement les données des visionneuses inconnues. Lors de l'utilisation d'une connexion Munchkin, les détails de l'utilisateur ne sont pas collectés. Pour obtenir des informations plus détaillées sur l'utilisateur, la connexion REST Marketo doit être utilisée. Pour plus d'informations, voir Intégration de Video Cloud à Marketo à l'aide des API REST.

Affichage des performances des prospects

Pour afficher les informations sur les pistes, cliquez sur l'onglet Leads. Notez que l'onglet Leads n'apparaîtra que si des formulaires de pistes de campagne ont été créés sur votre compte. La section Performances des prospects fournit des performances de haut niveau pour les vidéos qui ont été lues dans des lecteurs compatibles avec la campagne. Commencez par sélectionner une période pour les graphiques. Les options suivantes sont disponibles:

  • 7 jours
  • 30 jours
  • 90 jours

Les graphiques suivants s'affichent:

  • Entonnoir de campagne
  • Meilleures vidéos de conversion
  • Graphique Leads

Entonnoir de campagne

L'entonnoir de campagne affiche les statistiques des vidéos des lecteurs compatibles avec la campagne avec des formulaires de pistes de campagne pour la période sélectionnée:

  • Impressions vidéo: nombre de vidéos chargées
  • Vues de vidéos: nombre de fois qu'une vidéo a commencé à être lue, enregistré au démarrage du flux (n'inclut pas le retour en arrière ou la rediffusion) ; il ne s'agit pas d'une mesure des spectateurs individuels
  • Vues du formulaire de prospect - Le nombre total de vues de formulaire de prospect
  • Conversions - Le nombre total de formulaires de prospects qui ont été soumis

Meilleures vidéos de conversion

Le graphique Top Converting Videos affiche les vidéos ayant le taux de conversion le plus élevé. Le taux de conversion est calculé en divisant le nombre de formulaires de prospects soumis divisé par le nombre d'impressions vidéo.

Graphique Leads

Le graphique des pistes affiche le nombre de formulaires de prospects soumis par jour pour la période sélectionnée. Le survol du graphique affichera les points de données réels.

Affichage de l'état du module Campaign

La section Statut de la campagne affiche l'état actuel du module Campaign.

Configuration des connexions de données

L'option Connexions de données vous permet de configurer les intégrations Campaign. Campaign peut s'intégrer à des plateformes d'automatisation marketing populaires (intégration MAP-fr/) ou les données de visionnage peuvent être enregistrées localement, puis exportées ou extraites à l'aide de l'API Audience (intégration non-MAP-fr/). Le tableau suivant présente les intégrations prises en charge.

Intégrations MAP Intégrations non MAP
  • Oracle Eloqua
  • Marketo (API REST)
  • Marketo (Munchkin)
  • Salesforce
  • HubSpot (API REST)
  • HubSpot (API côté client)
  • Intégrations personnalisées
  • Authentification unique
  • Suivi des campagnes

Lorsque vous publiez des lecteurs compatibles avec la campagne, utilisez le code de publication intégré avancé (in-page). Le code de publication Standard Embed (iframe) ne fonctionnera pas car le plugin Campaign s'exécutera à l'intérieur d'une iframe et ne pourra donc pas accéder aux paramètres d'URL de la page parente et risque de ne pas recevoir les bons cookies.

Remarque: Le nombre de connexions pouvant être configurées dépend de votre abonnement Video Cloud.

Création de lecteur s de campagne

L'option Lecteurs de campagne vous permet de créer des lecteurs Brightcove compatibles avec la campagne. Les données d'engagement vidéo ne sont enregistrées que lorsque le contenu vidéo est visionné dans un lecteur vidéo compatible avec la campagne. Pour plus d'informations sur la création de lecteurs compatibles avec la campagne, consultez Création d'un lecteur compatible avec la campagne.

Création et gestion de formulaires de prospects

L'option Lead Forms vous permet de créer des formulaires de prospects pour la capture de données. Une fois qu'un formulaire de prospect est créé, il peut être associé à un lecteur compatible avec la campagne. Pour plus d'informations sur la création de formulaires de prospects, voir Création d'un formulaire de prospect de campagne. Il est également possible de créer et d'utiliser des formulaires de prospects créés à l'intérieur d'une plateforme tierce d'automatisation du marketing.

Publier une vidéo par e-mail

L'option Publier sur e-mail vous permet de sélectionner un lecteur, une vidéo, puis de générer le code HTML intégré pour envoyer la vidéo par e-mail. Pour plus d'informations, consultez Publier une vidéo par e-mail à l'aide du module Campaign.

Remarque: Le module Médias permet également de publier une vidéo par e-mail. Pour plus d'informations, consultez Publier une vidéo par e-mail.

Affichage des événements des lecteur s

L'option Toutes les activités récentes vous permet de visualiser les événements enregistrés au cours des dernières 24 heures. Pour plus d'informations, consultez la section Affichage des événements des lecteur s dans le module Campaign.

Affichage de l'activité vidéo

L'option Vidéo (non cartographique) vous permet de visualiser l'activité de visionnage d'une vidéo. Pour plus d'informations, voir Affichage de l'activité vidéo.

Affichage des profils des spectateurs

L'option Viewer Profile (non-map) vous permet de visualiser les événements de visualisation du lecteur par visionneuse. Cette option est activée lors de l'utilisation de l'authentification unique ou des intégrations personnalisées, ou du suivi des campagnes. Pour plus d'informations, consultez la section Affichage des informations sur le profil

Affichage des activités de synchronisation

L'option Synchronization Activity (Activité de synchronisation) affiche les 10 dernières tâches synchronisées de Campaign vers une plateforme d'automatisation du marketing. La synchronisation s'exécute toutes les heures. Sélectionnez une connexion, puis cliquez sur le bouton Tester la synchronisation pour exécuter une synchronisation. Cela est utile lorsque vous testez de nouvelles intégrations afin d'éviter d'avoir à attendre de voir si les données d'événements sont envoyées à votre plateforme d'automatisation du marketing. Test Sync peut être exécuté une fois toutes les 5 minutes jusqu'à 10 fois par heure.

Remarque: Lors de l'utilisation d'intégrations personnalisées, aucune tâche de synchronisation n'est créée. Les tâches de synchronisation sont créées uniquement lors de l'utilisation d'intégrations avec une plateforme d'automatisation du marketing.

Configuration des paramètres de campagne

L'option Paramètres vous permet de configurer certains paramètres par défaut pour le compte. Pour plus d'informations, consultez Configuration des paramètres de campagne.

Exporter les données de

L'option Exporter les données de campagne est utilisée pour exporter les données des formulaires d'événements et de prospects. Cette option ne s'affichera que pour les administrateurs de compte. Pour plus d'informations sur l'exportation des données de campagne, voir Exportation des données de campagne depuis le module Campaign

Exporter des prospects Gallery

L'option Exporter les pistes de la galerie est utilisée pour exporter les données de formulaire de prospects capturées à partir des expériences Gallery. Cette option ne s'affichera que pour les administrateurs de compte. Pour plus d'informations sur l'exportation des données de prospects Gallery, voir Exportation des données de prospects Gallery.

Gestion des données des spectateurs

L'option Gérer les données de visionneuse vous permet de rechercher un utilisateur, puis de le supprimer ou d'exporter les données de cet utilisateur. Cette option ne s'affichera que pour les administrateurs de compte. Pour plus d'informations sur la gestion des données de visionneuse, consultez la section Gestion des données
