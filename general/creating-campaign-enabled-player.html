Les données d'engagement vidéo ne seront enregistrées que lorsque le contenu vidéo est visionné dans un lecteur Brightcove compatible avec la campagne. Il existe deux manières de créer un lecteur compatible avec la campagne:

  • Créer un nouveau lecteur à l'aide du module Campaign
  • Activer la campagne sur un lecteur existant
Notez que quelle que soit la manière dont vous créez votre lecteur compatible avec la campagne, vous devez utiliser le code Avancé lorsque vous l'intégrez à une page.

Créer un nouveau lecteur à l'aide du module Campaign

Avant de créer un lecteur compatible avec la campagne à l'aide du module Campaign, le module Campaign doit être configuré avec au moins une connexion de données:

  • Eloqua
  • Marketo (Munchkin)
  • Marketo (API REST)
  • Salesforce
  • HubSpot (API REST)
  • HubSpot (API côté client)
  • SSO
  • Intégration personnalisée

Notez que le suivi des campagnes est fourni par défaut. Pour créer un lecteur compatible avec la campagne à l'aide du module Campagne, procédez comme suit.

  1. Ouvrez le module Campaign.
  2. Dans le menu de navigation de gauche, cliquez sur Joueurs de campagne.
  3. Cliquez sur Create Player. Un nouveau lecteur compatible avec la campagne sera créé et apparaîtra dans la liste des lecteur s.
  4. Dans la liste déroulante Se connecter à, sélectionnez la connexion de données que le lecteur utilisera.
  5. (Facultatif) Dans le menu déroulant Lead Form, sélectionnez le formulaire de lead que le lecteur utilisera.

Activation de la campagne sur un lecteur existant

Pour activer la campagne d'un lecteur existant, procédez comme suit.

  1. Ouvrez le module Players.
  2. Cliquez sur le nom d'un lecteur pour ouvrir ses propriétés.
  3. Cliquez sur Campagne dans le menu de navigation de gauche.
  4. Cochez l'option Activer la campagne.
  5. Utilisez la liste déroulante Se connecter à pour sélectionner une connexion à Campaign.
  6. (Facultatif) Utilisez le menu déroulant Lead Form pour sélectionner un formulaire de prospect à utiliser.
  7. Cliquez sur Enregistrer. Un message s'affiche pour vous indiquer que vous avez des modifications non publiées.
  8. Cliquez sur Publier et intégrer, puis sur Publier les modifications pour publier la modification sur le lecteur.
  9. Revenez au module Campaign. Cliquez sur Joueurs de campagne dans le menu de navigation de gauche.
  10. Vérifiez que le lecteur apparaît dans la liste des lecteur s de la campagne.
