Les codes UTM (également appelés paramètres UTM) sont de simples extraits de texte que vous pouvez ajouter à une URL pour suivre le succès de votre contenu Web. Il existe 5 paramètres standard que vous pouvez ajouter à n'importe quelle URL:

  • Campagne
  • Source
  • Moyen
  • Le contexte
  • Durée

Les codes UTM sont généralement ajoutés aux URL personnalisées. Vous trouverez ci-dessous un exemple de codes UTM (en gras) ajoutés à une URL.

https://www.brightcove.com/en/blog/2018/08/define-your-video-strategy-3-simple-steps-1 ? utm_campaign=Blogpost&UTM_MEDIUM=Social&UTM_SOURCE=LinkedIn

Les codes UTM vous permettent de suivre le fonctionnement d'un article de blog, d'une publicité Internet ou d'un e-mail sans avoir à créer des pages de destination personnalisées pour chaque campagne. En créant un code UTM distinct pour les articles de blog, les publicités et les e-mails, vous pouvez obtenir des données sur lesquelles génère plus de trafic, de conversions, etc.

Configuration des connexions de données pour qu'elles fonctionnent avec les champs UTM

Campaign analyse les champs UTM à partir des arguments d'URL. Vous pouvez ensuite accéder aux données UTM à partir de votre logiciel d'automatisation du marketing ou en utilisant l'API Audience. La configuration de la campagne requise est basée sur la plate-forme d'automatisation du marketing utilisée.

  • HubSpot
  • Salesforce
  • Eloqua
  • Marketo

HubSpot

Avec les connexions de données HubSpot, aucune configuration supplémentaire n'est requise. Les champs UTM s'affichent automatiquement.

Salesforce

Avec les connexions de données Salesforce, utilisez un type de mappage avancé, puis les champs UTM peuvent être mappés aux objets de données appropriés dans Salesforce.

Eloqua

Avec les connexions de données Eloqua, aucune configuration supplémentaire n'est requise. Les champs UTM s'affichent automatiquement.

Marketo

Comme il n'y a pas assez de champs que nous pouvons ajouter à l'activité personnalisée que nous créons dans Marketo, les champs UTM ne sont pas pris en charge avec les connexions Marketo.

Exporter des données en utilisant le module Campaign

Les champs UTM sont inclus lorsque vous exportez des données de campagne et que vous sélectionnez Afficher les événements. Pour plus d'informations sur l'exportation des données, voir Exportation des données de campagne depuis le module Campaign.

Accès aux champs UTM à l'aide de l'API Audience

Les champs UTM sont accessibles à l'aide de l'API Audience. Pour plus d'informations, consultez la documentation de l'API Audience
