Lorsque vous utilisez les intégrations Campaign Tracking, le module Campaign vous permet de sélectionner une vidéo, puis de visualiser l'activité associée à cette vidéo. Cela vous permet de voir qui a visionné une vidéo en particulier.

Toutes les vidéos visionnées dans les lecteurs compatibles avec la campagne feront l'objet d'un suivi. Pour afficher l'activité vidéo, cliquez sur Vidéo (non cartographique) dans le volet de navigation de gauche. La page Activité de visionnage affiche une liste de toutes les vidéos visionnées à partir des intégrations de suivi de Campaign pour la période sélectionnée. La commande Sélectionner une connexion peut être utilisée pour sélectionner une connexion spécifique. Le contrôle du calendrier peut être utilisé pour sélectionner une nouvelle plage de dates. Les données peuvent être triées en cliquant sur les en-têtes de colonne.

Remarques:
  • Il y aura un délai de quelques heures avant que les données de visionnage n'apparaissent.
  • Analytics et Campaign suivent via des plugins indépendants et des systèmes backend. Il est très fréquent de ne pas voir les mêmes résultats. Campaign respecte également des paramètres tels que Ne pas suivre-fr/ dans le navigateur. Pour obtenir les analyses les plus complètes, consultez les rapports du module Analytics.
  • L'activité vidéo n'est disponible que pour les connexions non-MAP (suivi de campagne).

Cliquez sur le lien d'une vidéo pour afficher toutes les activités de visionnage de cette vidéo. Les données peuvent être triées en cliquant sur les en-têtes de colonne.

Remarque: L'activité des spectateurs n'est disponible que pour les connexions non cartographiques (suivi de campagne).

Les données peuvent être exportées en cliquant sur les boutons CSV ou XLS en haut de la page.

Autres options d'affichage des données

Le module Campaign propose plusieurs méthodes pour afficher/exporter des données:

  • Activité récente: permet de visualiser les données des événements des dernières 24 heures. Pour plus d'informations, consultez la section Affichage des événements des lecteur s dans le module Campaign.
  • Profils de visionneuse: permet de visualiser l'activité vidéo par spectateur. Pour plus d'informations, consultez la section Affichage des informations sur le profil
  • Exporter les données - Utilisé pour exporter les données de visionnage de vidéos et de prospects. Pour plus d'informations, voir Exportation des données de campagne à partir du module Campaign.
  • API Audience - L'API Audience peut être utilisée pour récupérer par programme les données de visionnage de vidéos et de prospects. Pour plus d'informations, voir Vue d'ensemble: API Audience.
