Lors de l'utilisation des intégrations Campaign Tracking, les profils des spectateurs sont créés en fonction des données utilisées pour identifier le spectateur. Les profils de visionneuse vous permettent de visualiser les données de visionnage d'un spectateur donné.

Toutes les vidéos visionnées dans les lecteurs compatibles avec la campagne feront l'objet d'un suivi. Pour afficher l'activité vidéo, cliquez sur Profil de visionneuse (non cartographique) dans le volet de navigation de gauche. La page Profils des spectateurs affiche une liste de toutes les activités de visionnage issues des intégrations de suivi de Campaign pour la période sélectionnée. Le contrôle du calendrier peut être utilisé pour sélectionner une nouvelle plage de dates. La valeur Viewer sera celle qui a été utilisée pour identifier l'utilisateur. En général, il s'agit de l'adresse e-mail saisie sur un formulaire de prospect. Les données peuvent être triées en cliquant sur les en-têtes de colonne.

Remarque: Il y aura un délai de quelques heures avant que les données de visionnage n'apparaissent.

Cliquer sur le lien correspondant à une visionneuse affichera toutes les activités de visionnage de cette visionneuse. Les données peuvent être triées en cliquant sur les en-têtes de colonne.

Les données peuvent être exportées en cliquant sur les boutons CSV ou XLS en haut de la page.

Autres options d'affichage des données

Les données de campagne peuvent être visualisées/exportées de différentes manières:

  • Activité récente: permet de visualiser les données des événements des dernières 24 heures. Pour plus d'informations, consultez la section Affichage des événements des lecteur s dans le module Campaign.
  • Activité vidéo: permet de visualiser l'activité des spectateurs par vidéo. Pour plus d'informations, voir Affichage de l'activité vidéo.
  • Exporter les données - Utilisé pour exporter les données de visionnage de vidéos et de prospects. Pour plus d'informations, voir Exportation des données de campagne à partir du module Campaign.
  • API Audience - L'API Audience peut être utilisée pour récupérer par programme les données de visionnage de vidéos et de prospects. Pour plus d'informations, voir Vue d'ensemble: API Audience.
